## About Open GeoScience

Open GeoScience bridges the OSGeo community with researchers in Earth sciences, environmental science, and related fields. We advocate for:

{{< columns-start >}}
{{< column-start class="is-4" >}}
{{< rich-box-start icon="fa-solid fa-code" >}}
### Open Source

Using and contributing to open source geospatial software for research workflows.
{{< rich-box-end >}}
{{< column-end >}}

{{< column-start class="is-4" >}}
{{< rich-box-start icon="fa-solid fa-database" >}}
### Open Data

{{< rich-box-end >}}
{{< column-end >}}

{{< column-start class="is-4" >}}
{{< rich-box-start icon="fa-solid fa-file-lines" >}}

## Activities

The Open GeoScience committee works on:

- **Advocacy** - Promoting open practices in geoscience communities
- **Workshops** - Organizing training on OSGeo tools for researchers
- **Collaboration** - Connecting with scientific organizations
- **Documentation** - Creating resources for geoscience workflows
