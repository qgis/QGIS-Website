### Recent Major Releases

{{< rich-box-start icon="fa-solid fa-rocket" >}}
#### QGIS 3.40 LTR

*October 2024*

The latest Long Term Release of QGIS brings improved performance, new processing algorithms, and enhanced 3D visualization capabilities.

**Highlights:**
- New mesh layer rendering options
- Enhanced GPS tracking functionality
- Improved annotation tools
- Performance optimizations

[Download QGIS](https://qgis.org/download/) | [Release Notes](https://qgis.org/en/site/forusers/visualchangelog340/)
{{< rich-box-end >}}

{{< rich-box-start icon="fa-solid fa-rocket" >}}
#### GDAL 3.9

*2024*

The geospatial data abstraction library continues to evolve with new formats, performance improvements, and cloud-native capabilities.

**Highlights:**
- New format drivers
- Improved cloud access
- Performance enhancements
- Bug fixes and stability

[Download GDAL](https://gdal.org/download.html) | [Release Notes](https://github.com/OSGeo/gdal/releases)
{{< rich-box-end >}}

{{< rich-box-start icon="fa-solid fa-rocket" >}}
#### PostGIS 3.4

*2024*

Spatial database functionality for PostgreSQL with new features and improved standards support.

**Highlights:**
- New spatial functions
- Performance improvements
- Standards compliance updates
- PostgreSQL 16 support

[Download PostGIS](https://postgis.net/install/) | [Release Notes](https://postgis.net/docs/manual-3.4/release_notes.html)
{{< rich-box-end >}}

{{< rich-box-start icon="fa-solid fa-rocket" >}}
#### GeoServer 2.25

*2024*

Java-based software server for sharing and editing geospatial data.

**Highlights:**
- OGC API improvements
- Security enhancements
- New styling options
- Performance tuning

[Download GeoServer](https://geoserver.org/release/stable/) | [Release Notes](https://geoserver.org/announcements/)
{{< rich-box-end >}}

<h2 class="section-title">Release Schedules</h2>
</div>

OSGeo projects follow different release strategies:

### Point Releases
Regular bug fix and security releases, typically every few weeks.

### Feature Releases
New functionality releases, typically every few months.

### Long Term Releases (LTR)
Stable releases maintained for extended periods (1-3 years) with backported bug fixes.
