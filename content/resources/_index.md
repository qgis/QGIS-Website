<h2 class="section-title">Resources</h2>
</div>

OSGeo provides a variety of resources to help you learn about and use open source geospatial software.

### OSGeoLive

OSGeoLive is a self-contained bootable DVD, USB thumb drive, or Virtual Machine based on Lubuntu that allows you to try a wide variety of open source geospatial software without installing anything.

[Learn about OSGeoLive](/resources/live-dvd/)

### Documentation

Each OSGeo project maintains its own documentation. Visit individual project websites for detailed user guides, API documentation, and tutorials.

### Education

Through our Geo for All initiative, OSGeo supports geospatial education worldwide. Find educational resources, course materials, and training opportunities.

[Education Resources](/resources/education/)

### Events

Stay updated on upcoming conferences, workshops, and meetups in the open source geospatial community.

[View Events](/events/)

### News

Keep up with the latest developments in the OSGeo community.

[Read News](/news/)

### Mailing Lists

The primary way to get help with OSGeo projects is through project mailing lists. Community members are happy to help answer questions and solve problems.

[View Mailing Lists](/community/mailing-lists/)

### Commercial Support

If you need professional support, training, or consulting services, many companies offer commercial support for OSGeo projects.

### Local Chapters

Connect with your local OSGeo community for in-person help and support.

[Find Local Chapters](/local-chapters/)
