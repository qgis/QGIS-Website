<h2 class="section-title">Get Involved</h2>
</div>

There are many ways to contribute to the OSGeo community, regardless of your background or skill level:

{{< columns-start >}}

{{< column-start class="is-6" >}}

### For Users
- **Use** open source geospatial software and share your experiences
- **Report bugs** when you find them
- **Suggest features** that would help your work
- **Write reviews** and share case studies
- **Help others** on mailing lists and forums
- **Spread the word** about open source GIS

{{< column-end >}}

{{< column-start class="is-6" >}}

### For Developers
- **Fix bugs** and submit pull requests
- **Write documentation** and tutorials
- **Translate** interfaces and documentation
- **Review code** from other contributors
- **Mentor** new contributors
- **Participate** in Google Summer of Code

{{< column-end >}}

{{< columns-end >}}

{{< columns-start >}}

{{< column-start class="is-6" >}}

### For Educators
- **Use** OSGeo software in courses
- **Contribute** to Geo for All
- **Share** teaching materials
- **Present** at conferences
- **Establish** a Geo for All lab

{{< column-end >}}

{{< column-start class="is-6" >}}

### For Organizations
- **Adopt** open source geospatial software
- **Become a sponsor** of OSGeo
- **Register** as a service provider
- **Host** local chapter events
- **Contribute** developer time

{{< column-end >}}

{{< columns-end >}}

[Full Getting Involved Guide &rarr;](/community/getting-started/)

<h2 class="section-title">Code of Conduct</h2>
</div>

OSGeo is committed to providing a **welcoming and inclusive environment** for all community members. We expect all participants to:

{{< columns-start >}}

{{< column-start class="is-6" >}}

- **Be respectful** of different viewpoints and experiences
- **Use welcoming** and inclusive language
- **Be collaborative** and constructive
- **Focus on** what's best for the community
- **Show empathy** towards other community members

{{< column-end >}}

{{< column-start class="is-6" >}}
{{< rich-box-start >}}
### Report an Issue

If you experience or witness behavior that violates our Code of Conduct, please report it to the Code of Conduct Committee.

[Read Full Code of Conduct &rarr;](/community/code-of-conduct/)

[Report an Issue &rarr;](/community/code-of-conduct/report/)
