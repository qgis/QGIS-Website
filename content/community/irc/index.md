## Real-Time Chat

While mailing lists are great for discussions that don't require immediate response, sometimes you need real-time help or just want to chat with the community. OSGeo projects use several chat platforms.

## IRC on Libera.Chat

Many OSGeo projects have IRC channels on [Libera.Chat](https://libera.chat/).

### How to Connect

1. **Web client**: Visit [web.libera.chat](https://web.libera.chat/)
2. **Desktop client**: Use a client like HexChat, irssi, or weechat
3. **Server**: `irc.libera.chat` (port 6667, or 6697 for SSL)

### OSGeo Channels

| Channel | Project |
|---------|---------|
| `#osgeo` | General OSGeo discussion |
| `#qgis` | QGIS Desktop |
| `#grass` | GRASS GIS |
| `#geoserver` | GeoServer |
| `#postgis` | PostGIS |
| `#gdal` | GDAL/OGR |
| `#openlayers` | OpenLayers |
| `#mapserver` | MapServer |

## Matrix

Matrix is a modern, federated chat protocol. Many IRC channels are bridged to Matrix, so you can use either.

### Joining via Matrix

1. Create an account on [matrix.org](https://matrix.org) or another Matrix server
2. Use a client like [Element](https://element.io/)
3. Join rooms like `#qgis:matrix.org`

<div class="notification is-info is-light">
<strong>Tip:</strong> Matrix rooms bridged to IRC show messages from both platforms, so you can participate from either.
</div>

## Chat Etiquette

<div class="box has-background-light">
<h4>Tips for Getting Help</h4>
<ul>
<li><strong>Ask your question directly</strong> - Don't ask "Can anyone help?" Just ask!</li>
<li><strong>Be patient</strong> - People are in different time zones</li>
<li><strong>Provide context</strong> - Include software version, OS, and what you've tried</li>
<li><strong>Use paste services</strong> - For code/logs, use sites like pastebin.com</li>
<li><strong>Stay on topic</strong> - Use the appropriate channel for your question</li>
<li><strong>Be respectful</strong> - Follow the <a href="/community/code-of-conduct/">Code of Conduct</a></li>
</ul>
</div>

For most users, we recommend:

1. **Start with IRC** via the web client at [web.libera.chat](https://web.libera.chat/)
2. **Join #osgeo** for general questions
3. **Join your project's channel** for specific help
