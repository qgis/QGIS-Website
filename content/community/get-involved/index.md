<h2 class="section-title">Why Contribute?</h2>
</div>

Open source geospatial software is built by people like you. Contributing helps:

- **Improve software** you use daily
- **Build skills** and gain experience
- **Connect** with a global community
- **Give back** to projects that have helped you
- **Shape the future** of geospatial technology

<div class="section-header">
<div class="section-header-badge">
<i class="fa-solid fa-hands-helping"></i>
<span>Options</span>
</div>
<h2 class="section-title">Ways to Contribute</h2>
</div>

You don't need to be a programmer to contribute! Here are many ways to help:

### Documentation

<div class="box has-background-light">
<div class="columns is-vcentered">
<div class="column is-narrow">
<span class="icon is-large has-text-success">
<i class="fa-solid fa-book fa-3x"></i>
</span>
</div>
<div class="column">
<h4 class="mb-2">Write and Improve Documentation</h4>
<p>Good documentation helps everyone. You can:</p>
<ul>
<li>Fix typos and clarify explanations</li>
<li>Write tutorials based on your experience</li>
<li>Update outdated information</li>
<li>Add examples and screenshots</li>
</ul>
</div>
</div>
</div>

### Translation

<div class="box has-background-light">
<div class="columns is-vcentered">
<div class="column is-narrow">
<span class="icon is-large has-text-success">
<i class="fa-solid fa-language fa-3x"></i>
</span>
</div>
<div class="column">
<h4 class="mb-2">Translate Software and Documentation</h4>
<p>Help make software accessible to speakers of other languages:</p>
<ul>
<li>Translate user interfaces</li>
<li>Translate documentation</li>
<li>Review existing translations</li>
</ul>
</div>
</div>
</div>

### Testing

<div class="box has-background-light">
<div class="columns is-vcentered">
<div class="column is-narrow">
<span class="icon is-large has-text-success">
<i class="fa-solid fa-bug fa-3x"></i>
</span>
</div>
<div class="column">
<h4 class="mb-2">Test and Report Bugs</h4>
<p>Quality assurance is crucial:</p>
<ul>
<li>Test new releases before they're official</li>
<li>Report bugs with clear reproduction steps</li>
<li>Verify that bug fixes actually work</li>
<li>Test on different platforms and configurations</li>
</ul>
</div>
</div>
</div>

### Support

<div class="box has-background-light">
<div class="columns is-vcentered">
<div class="column is-narrow">
<span class="icon is-large has-text-success">
<i class="fa-solid fa-comments fa-3x"></i>
</span>
</div>
<div class="column">
<h4 class="mb-2">Help Others</h4>
<p>Share your knowledge:</p>
<ul>
<li>Answer questions on mailing lists</li>
<li>Help users on Stack Exchange</li>
<li>Mentor newcomers</li>
<li>Create video tutorials</li>
</ul>
</div>
</div>
</div>

### Code

<div class="box has-background-light">
<div class="columns is-vcentered">
<div class="column is-narrow">
<span class="icon is-large has-text-success">
<i class="fa-solid fa-code fa-3x"></i>
</span>
</div>
<div class="column">
<h4 class="mb-2">Write Code</h4>
<p>If you're a developer:</p>
<ul>
<li>Fix bugs (great way to start!)</li>
<li>Implement new features</li>
<li>Create plugins and extensions</li>
<li>Improve performance</li>
<li>Review others' code</li>
</ul>
</div>
</div>
</div>

### Outreach

<div class="box has-background-light">
<div class="columns is-vcentered">
<div class="column is-narrow">

<div class="column">
<h4 class="mb-2">Spread the Word</h4>

<h2 class="section-title">How to Start</h2>
</div>

<div class="content-panel is-featured">
<div class="content-panel-header">
<div class="content-panel-icon">
<i class="fa-solid fa-shoe-prints"></i>
</div>
<div>
<h4 class="content-panel-title">Your First Steps</h4>
<p class="content-panel-subtitle">A simple path to contribution</p>
</div>
</div>
<div class="content-panel-body">
<ol class="numbered-steps">
<li><strong>Choose a project</strong> - Start with software you already use</li>
<li><strong>Read the contribution guidelines</strong> - Most projects have a CONTRIBUTING file</li>
<li><strong>Join the mailing list</strong> - Introduce yourself and ask questions</li>
<li><strong>Start small</strong> - Fix a typo, report a bug, or answer a question</li>
<li><strong>Be patient</strong> - Everyone is learning and improving together</li>
</ol>
</div>
</div>
