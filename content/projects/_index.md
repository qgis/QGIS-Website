### QGIS Desktop

<span class="category-tag is-desktop">Desktop GIS</span>

The leading Free and Open Source Desktop GIS. Create, edit, visualize, analyze, and publish geospatial information on Windows, Mac, Linux, and BSD.

**Key Features:**
- Professional cartography and map production
- Advanced vector and raster analysis
- 3D visualization
- Python scripting and plugin ecosystem
- Mobile companion app (QField)

[Visit QGIS.org](https://qgis.org) | [Download](https://qgis.org/download/)

**Key Features:**
- Advanced raster and vector analysis
- Image processing and classification
- Temporal data analysis
- Hydrological modeling
- Integration with QGIS and R

[Visit GRASS GIS](https://grass.osgeo.org) | [Download](https://grass.osgeo.org/download/)

### GeoServer

<span class="category-tag is-web">Web Mapping</span>

Share, process, and edit geospatial data. Designed for interoperability, it publishes data from any major spatial data source using open standards.

**Key Features:**
- WMS, WFS, WCS, WMTS support
- REST API
- Tile caching integration
- Security and authentication
- Vector tiles

[Visit GeoServer](https://geoserver.org) | [Download](https://geoserver.org/release/stable/)

## Geospatial Libraries

Core libraries that power geospatial applications.

{{< columns-start >}}

{{< column-start class="is-4" >}}
{{< rich-box-start >}}
### GDAL/OGR

<span class="category-tag is-library">Data Translation</span>

The Geospatial Data Abstraction Library - translator library for raster and vector geospatial data formats.

**Key Features:**
- 200+ raster formats
- 100+ vector formats
- Command line utilities
- Python bindings
- Cloud-optimized formats

[Visit GDAL](https://gdal.org) | [Documentation](https://gdal.org/documentation.html)
{{< rich-box-end >}}
{{< column-end >}}

{{< column-start class="is-4" >}}
{{< rich-box-start >}}
### PROJ

<span class="category-tag is-library">Coordinate Transformation</span>

Cartographic projections and coordinate transformations library. The standard for geospatial coordinate operations.

**Key Features:**
- Thousands of CRS definitions
- High-precision transformations
- Datum transformations
- Command line tools
- EPSG database

[Visit PROJ](https://proj.org) | [Documentation](https://proj.org/documentation.html)
{{< rich-box-end >}}
{{< column-end >}}

{{< column-start class="is-4" >}}
{{< rich-box-start >}}
### GEOS

<span class="category-tag is-library">Geometry Engine</span>

Geometry Engine Open Source - C++ implementation of spatial predicates and operations.

**Key Features:**
- JTS port to C++
- Spatial predicates
- Geometric operations
- Spatial indexing
- Used by PostGIS, QGIS, etc.

[Visit GEOS](https://libgeos.org) | [Documentation](https://libgeos.org/usage/c_api/)

## Spatial Databases

Store, query, and analyze spatial data at scale.

{{< columns-start >}}

{{< column-start class="is-6" >}}
{{< rich-box-start >}}
### PostGIS

<span class="category-tag is-database">PostgreSQL Extension</span>

Spatial database extender for PostgreSQL. Adds support for geographic objects allowing location queries to be run in SQL.

**Key Features:**
- Geometry and geography types
- Spatial indexing (GiST, SP-GiST)
- Raster data support
- Topology
- Point cloud support
- 3D and 4D coordinates

[Visit PostGIS](https://postgis.net) | [Documentation](https://postgis.net/documentation/)
