## What is Incubation?

OSGeo's incubation process is the pathway for open source geospatial projects to become officially recognized as OSGeo projects. The process ensures that projects meet community standards for openness, quality, and sustainability.

## Incubation Criteria

Projects are evaluated on:

### Open Source License
- Must use an OSI-approved open source license
- All dependencies must be compatible

### Community
- Open and welcoming community
- Clear contribution guidelines
- Responsive to bug reports and questions
- Multiple active contributors

### Documentation
- User documentation
- Developer documentation
- Installation guides
- API documentation (if applicable)

### Code Quality
- Version control (typically Git)
- Issue tracking
- Quality assurance practices
- Regular releases

### Governance
- Clear decision-making process
- Project steering committee or equivalent
- Transparent communication

### Project Levels

**OSGeo Project**
Fully graduated, meets all criteria

**Incubating Project**
In the process of meeting criteria

**Community Project**
Associated but not governed by OSGeo
{{< box-end >}}

The committee oversees the incubation process:

- Reviews applications
- Assigns mentors
- Evaluates progress
- Recommends graduation
