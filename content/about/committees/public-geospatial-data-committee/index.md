<h2 class="section-title">Committee Goals</h2>
</div>

The committee pursues these core goals:

{{< columns-start >}}

{{< column-start class="is-6" >}}

{{< rich-box-start icon="fa-solid fa-file-code" >}}
### Advance Open Formats

Promote adoption of open geospatial formats that enable interoperability and data sharing.
{{< rich-box-end >}}

{{< rich-box-start icon="fa-solid fa-database" >}}
### Expand Public Geodata

Work to expand public availability of government-collected geodata.
{{< rich-box-end >}}

{{< column-end >}}

{{< column-start class="is-6" >}}

{{< rich-box-start icon="fa-solid fa-archive" >}}
### Maintain Open Archive

Maintain an open geodata archive for community use.
{{< rich-box-end >}}

{{< rich-box-start icon="fa-solid fa-scale-balanced" >}}
### Clarify Licensing

Clarify licensing frameworks for public geodata.
{{< rich-box-end >}}

{{< column-end >}}
