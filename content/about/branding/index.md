### Logo Usage Rules

<div class="box has-background-light">
<div class="columns">
<div class="column">
<h4 class="has-text-success"><i class="fa-solid fa-check mr-2"></i>Do</h4>
<ul>
<li>Use official logo files</li>
<li>Maintain aspect ratio</li>
<li>Ensure adequate clear space</li>
<li>Use appropriate color version</li>
</ul>
</div>
<div class="column">
<h4 class="has-text-danger"><i class="fa-solid fa-times mr-2"></i>Don't</h4>
<ul>
<li>Stretch or distort the logo</li>
<li>Change logo colors</li>
<li>Add effects or shadows</li>
<li>Place on busy backgrounds</li>
</ul>
</div>
</div>
</div>

<h2 class="section-title">Typography</h2>
</div>

OSGeo uses clean, readable typefaces:

- **Headings**: System fonts (San Francisco, Segoe UI, Roboto)
- **Body**: System fonts for optimal readability
- **Code**: Monospace fonts

- Conference presentations
- Project documentation
