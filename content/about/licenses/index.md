<h2 class="section-title">Contributor License Agreements</h2>
</div>

OSGeo projects accepting contributions use procedures tailored to individual initiatives. Two primary contributor agreements are available, derived from the Apache code contribution licenses:

{{< columns-start >}}

{{< column-start class="is-6" >}}

{{< rich-box-start icon="fa-solid fa-user" >}}
### Individual Contributor License Agreement V1.0

For individual contributors submitting code or documentation to OSGeo projects.

**Downloads:**
- [Text Format](/resources/individual-contributor-license/)
- [PDF Format](/resources/individual-contributor-license.pdf)

{{< rich-box-end >}}

{{< column-end >}}

{{< column-start class="is-6" >}}

{{< rich-box-start icon="fa-solid fa-building" >}}
### Software Grant and Corporate Contributor License Agreement V1.0

For organizations contributing code or granting software to OSGeo projects.

**Downloads:**
- [Text Format](/resources/corporate-contributor-license/)
- [PDF Format](/resources/corporate-contributor-license.pdf)
