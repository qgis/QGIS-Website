# サステイニングメンバーシッププログラム
QGISはオープンソースプロジェクトです。QGISの開発、ドキュメント、ウェブサイト、およびインフラは、主にボランティアによって運営されています。2002年のプロジェクト発足以来、誰もが自由に使用・改善できる、大規模で価値ある有用なコードベースを築き上げてきました。

私たちはウェブサイトでQGISのソースコードとバイナリを無償で提供しています。このボランティア主導のアプローチによって多大な成果を上げてきましたが、活動を継続するためにはさらなる資金援助が必要です
## 資金の使い方
集まった資金は、主に以下の目的に使用されています:
* バグ修正のための開発者への報酬
* QGISプロジェクトの大幅な機能改善を支援する年次グラントプログラムへの資金提供
* サーバーインフラの運用費用
* QGISが利用する上流ライブラリの改善への支援
* フルタイムのウェブインフラ管理者
* ドキュメント作成の専任担当者

今後は資金が許す限り、これらの活動をさらに広げていきたいと考えています。

サステイニングメンバーシップの有効期間は1年間です。期間終了後は、新たな支払いにより更新するか、そのまま失効させることができます。なお、サステイニングメンバーシップに議決権は含まれず、既存の議決権にも影響はありません。

会費はQGIS総会にて承認・改定されます。各カテゴリーの最低金額として定められていますが、サステイニングメンバーはご希望により最低額を超える金額をお支払いいただくことも可能です。

QGIS.ORGはグローバルなプロジェクトであり、企業・NGO・政府機関によって財政状況が異なるため、メンバーシップレベルの指定は行っていません。サステイニングメンバーは、自身の財政状況に応じたレベルを自由にお選びいただけます。

維持会員の承認は理事会の裁量と承認に従うものとし、理事会は必要に応じて会員維持と継続的支持会員の申請を承認または拒否する権利を留保します。
## Sustaining membership benefits
QGISプロジェクトを維持会員として支援することにより、以下の利益があります:

#### Public Acknowledgment
すべての維持館員メンバーおよび寄付者は、プロジェクトの [メンバーページ]({{< ref "/funding/membership.md" >}}) にメンバーシップカテゴリー順で掲載され、各組織・企業のウェブサイトへのリンクおよびカテゴリーに応じたサイズのロゴが表示されます。

#### ソフトウェア品質の向上
サステイニングメンバーシップからの支援は、バグや問題の迅速な解決に役立ち、より良いユーザーエクスペリエンスの実現につながります。{{< rich-content-end >}}{{< rich-box-end >}}{{< column-end >}}{{< column-start>}}

{{< rich-box-start >}} {{< rich-content-start themeClass="coloring-4 rich-content-text" >}}
#### ブランド認知度の向上
フラッグシップ・ラージ・ミディアムのサステイニングメンバーは、メンバーシップカテゴリーに応じたサイズと掲載順で、QGISホームページのフッターバナーにロゴが表示されます。

{{< rich-content-end >}} {{< rich-box-end >}} {{< column-end >}} {{< columns-end >}} {{< column-end >}} {{< columns-end >}}

{{< columns-start >}} {{< column-start>}}

{{< rich-box-start >}} {{< rich-content-start themeClass="coloring-4">}}
#### ソフトウェアアップデートへの掲載
サステイニングメンバーは、QGISソフトウェアリリースのビジュアルチェンジログにて紹介され、プロジェクトへの継続的な支援への取り組みが称えられます。 {{< rich-content-end >}}{{< rich-box-end >}}{{< column-end >}}{{< column-start>}}

{{< rich-box-start >}} {{< rich-content-start themeClass="coloring-4" >}}
#### Public Acknowledgment
プロジェクトへの支援の証として、特別バージョンのQGISロゴをウェブサイトや各種資料にご使用いただけます。

{{< rich-content-end >}} {{< rich-box-end >}} {{< column-end >}} {{< column-start>}}

{{< rich-box-start >}} {{< rich-content-start themeClass="coloring-4" >}}
#### QGISを広告なしに保つ
サステイニングメンバーシップによる支援のおかげで、広告などユーザーにとって好ましくない手段に頼ることなく、プロジェクトの運営資金を確保することができます。

{{< rich-content-end >}} {{< rich-box-end >}} {{< column-end >}} {{< columns-end >}}

資金の使途はQGISプロジェクト運営委員会が決定し、予算案としてQGIS.ORG総会に提出され、承認または否決されます。

{{< columns-start >}} {{< column-start class="is-flex-direction-column is-one-quarter">}}
#### 認定と普及活動
{{<column-end >}}

{{<column-start class="is-flex-direction-column is-align-items-flex-start is-three-quarters " >}}

**認定**:サステイニングメンバーはQGISウェブサイトに掲載され、フラッグシップ・ラージ・ミディアムメンバーのロゴがフッターバナーに表示されます。

**認知度向上**: メンバーはQGISリリースのビジュアルチェンジログに掲載されます。

**プロモーション**: 必要に応じて、QGISはサステイニングメンバーからの支援を紹介するソーシャルメディアでのプロモーションを行います。

{{< column-end >}} {{< columns-end >}}

##### 一回払い
寄付は金額を問わず、銀行口座への直接振込（SEPA送金）、クレジットカード決済（[stripe.com](http://stripe.com)経由）、またはPayPalでお受けしています（詳細は[寄付]({{< ref "/funding/donate.md" >}}) ページをご確認ください）。少額の寄付でもプロジェクトの重要な目標達成に大きく貢献できます！ {{< rich-content-end >}} {{< rich-box-end >}}

{{< rich-box-start icon="💁" layoutClass="tips">}} {{< rich-content-start themeClass="coloring-1" >}}
##### 「使途指定」資金提供について
「使途指定」による資金提供は管理上の問題が多く生じるため、大変申し訳ございますがお受けしておりません。QGISの開発における特定の分野への支援をご希望の場合は、開発者の雇用やバグ修正イニシアティブへの参加など、プロジェクトへの人的リソースの提供という形でのご支援をお勧めします。{{< rich-content-end >}}{{< rich-box-end >}}

{{< rich-box-start icon="⚠️" layoutClass="tips">}} {{< rich-content-start themeClass="coloring-6" >}}
##### 警告
サステイニングメンバーシップ申請の受否は、QGIS PSCの判断に委ねられています。申請組織またはその活動がプロジェクトの理念と相容れないと判断される場合、申請をお断りする権利を留保します。例えば、GIS・空間情報サービスと無関係のマーケティング会社からの申請や、ロゴやマーケティング資料の掲載がユーザー体験を損なうと判断される場合は、申請をお断りする場合があります。 {{< rich-content-end >}}{{< rich-box-end >}}
