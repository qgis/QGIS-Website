# エグゼクティブ・サマリー
QGISは、世界的に大きな影響力を持つプロジェクトです。QGIS.orgは、空間データの可視化、作成、編集、分析、および情報の公開を行うための、クラス最高のツールへのオープンなアクセスを提供しています。QGIS.orgはツールの使用方法を規定しませんが、より公正な社会の育成、より良い意思決定、そしてより持続可能な環境という目標を推進するために、誰もが自由にこれらのツールを利用できるようにしています。私たちは、従来のデスクトップPC、モバイルデバイス、そしてクラウドで動作するツールを提供しています。

## QGISを使っているのは誰？
QGISは、政府機関、多国籍大企業、法人、学術機関から、小規模企業や趣味で利用する個人ユーザーまで、幅広く利用されています。私たちのオープンソース開発モデルは、これらすべてのユーザーが負担可能なコストでソフトウェアを利用できるようにし、誰もがQGISの恩恵を受けられるようにしています。

{{<button-bar "fa-solid fa-download:License:license/" >}}
## QGISは無料で利用できますが、維持コストがかからないわけではありません
QGIS.orgが提供するような高品質なツールを開発するには、これまでも、そしてこれからも多大な努力が必要です。当初、このコストは献身的なボランティアの努力のみによって賄われてきました。プロジェクトが成熟するにつれ、プロジェクトを中心に経済的なエコシステムが形成されました。クライアント（利用者）は、自身のニーズをより良く満たすために、コミュニティのメンバーと契約してQGISプロジェクト (ソフトウェア、ドキュメント、オンライン・インフラストラクチャ) を拡張し始めたのです。

こうした直接的な資金調達活動と並行して、寄付や持続的会員制度を通じて、私たちの裁量で運用できる資金でプロジェクトを支援するプログラムを構築しました。これらの自由裁量資金により、プロジェクトを取り巻くすべてのサポートインフラを提供することが可能になります。さらに、この資金は以下の活動の源泉となっています。
- ソフトウェアの品質向上
- 開発者会議の開催
- プロジェクトの広範な要素を改善するための技術協力者の雇用、および
- 長期的な持続可能性を強化するための、プロジェクト周辺コミュニティの育成

## 「価値には価値を」のモデル
多くのユーザーが、組織内での利用を継続するにはコスト負担が大きすぎる商用ソフトウェアから、QGISへと移行しています。私たちは、QGISのような空間情報ツールを必要としながらも、以前は（費用の壁により）その門戸を閉ざされていた人々に対して、新たな道を開けることを誇りに思っています。しかし同時に、QGISへの移行によって得られた価値の一部を、私たちと共有していただくこともお願いしています。皆さまから提供された資金は、すべてのユーザーに直接的な利益をもたらすよう活用され、より機能が豊富で、安定しており、ドキュメントの充実したQGISツールキットの提供へと繋がります。この「価値の還元（Value for Value）」モデルは、QGISプロジェクトの長期的な持続可能性と発展を支える助けとなります。毎日約50万人近いアクティブユーザーがいる中で、皆さまの貢献は多くの人々の生活に大きな影響を与えることができます。本プロジェクトは、世界中の多くの著名な国際機関から現物および資金による寄付を受けており、深く感謝しています。私たちの活動に価値を見出し、活動を支援するための資金提供という形でその価値を還元してくださる組織の輪に、ぜひ皆さまも加わってください。

## ガバナンス
QGIS.orgはスイスの協会（Verein）として登録されており、非営利で運営されています。選出された理事会によって管理され、世界中の何百人もの人々によって貢献されています。私たちが開発するソフトウェアがオープンソースであるのと同様に、プロジェクトも「公開」という同じ原則に基づいて運営されており、すべての財務報告、理事会報告、コミュニティの決定は、オープンかつ透明性の高い方法で実行されています。
