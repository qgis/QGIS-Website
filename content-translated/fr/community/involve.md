### Développer le noyau de QGIS
QGIS Core comprend des bibliothèques permettant de développer des applications personnalisées grâce à une API robuste. Il inclut également des applications de bureau et serveur qui mettent en avant les capacités de ces bibliothèques, offrant une interface conviviale

[Contributor Requirements]({{< ref "resources/support/contributors-requirements" >}})

[Get set up for QGIS Core development](https://docs.qgis.org/latest/en/docs/developers_guide/index.html )

[QGIS roadmap]({{< ref "resources/roadmap" >}})
