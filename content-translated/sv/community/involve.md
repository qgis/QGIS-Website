### Utveckla QGIS Core
QGIS Core består av bibliotek för att bygga anpassade applikationer via ett robust API. Det innehåller även skrivbords- och serverapplikationer som visar upp bibliotekets funktioner och ger ett användarvänligt gränssnitt

[Krav på bidragsgivare]({{< ref "resources/support/contributors-requirements" >}})

[Gör dig redo för utveckling av QGIS Core](https://docs.qgis.org/latest/en/docs/developers_guide/index.html )
