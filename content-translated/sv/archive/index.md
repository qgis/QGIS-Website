# Arkiv
Tidigare versioner av QGIS finns tillgängliga på följande platser:
* [QGIS.org-hostade nedladdningar](/downloads)
* [Äldre utgåvor för macOS / OS X](/downloads/macOS/)
* [OSGeo-hostade nedladdningar](https://download.osgeo.org/qgis/)
* [Föråldrade 'Kyngchaos' macOS / OS X-nedladdningar](https://www.kyngchaos.com/software/archive/).

Insticksprogram för QGIS finns också tillgängliga [på plugins.qgis.org] (https://plugins.qgis.org/plugins/).
