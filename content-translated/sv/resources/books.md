# Books
På den här sidan listas böcker som finns tillgängliga från andra förlag. QGIS hanterar inte någon av dem, och de listas här endast för din bekvämlighet.

Skriv till [psc@qgis.org](mailto:psc%40qgis.org) om du vill att din bok ska läggas till här. Alternativt kan du redigera den här sidan på GitHub och skicka in en pull request.

Obs: Böckerna är listade i ordning med de nyaste först. Efter de engelska böckerna finns det artiklar på franska, grekiska, polska och andra språk.
