## 核心功能
{{<tabs tab1="创建地图" tab2="编辑图层" tab3="处理与分析" tab4="分享地图" >}}

{{< columns-start id="content-tab-1" >}} {{< column-start class="is-flex-direction-column is-two-thirds rounded mb-4" >}} ![创建地图](../img/create.gif "Create maps") {{< column-end >}}

{{<column-start class="is-flex-direction-column is-justify-content-center is-one-third" >}}
###### 业界顶级制图软件
体验QGIS丰富的设计选项，让您的制图创意落地成真。
###### 专业地图制作
借助直观的布局设计工具，制作高品质、大尺寸的打印地图。
###### 强大的报告工具
使用地图、介质和表格内容创建图集和报告。 {{< column-end >}} {{< columns-end >}}

{{< columns-start id="content-tab-2" >}} {{< column-start class="is-flex-direction-column is-two-thirds rounded mb-4" >}} ![编辑地图](../img/edit.gif "Edit map") {{< column-end >}}

{{<column-start class="is-flex-direction-column is-justify-content-center is-one-third" >}}
###### 卓越的数字化功能
精确创建与编辑点、线、多边形和网孔。
###### 高级构建工具
使用曲线、类CAD工具以及广泛的几何图形变换，快速构建几何图形。
###### 高度可定制的表单
借助表单设计器，轻松创建简洁易用的表单。

{{< column-end >}} {{< columns-end >}}

{{< columns-start id="content-tab-3" >}} {{< column-start class="is-flex-direction-column is-two-thirds rounded mb-4" >}} ![可视化地图](../img/visualize.gif "visualize map") {{< column-end >}}

{{<column-start class="is-flex-direction-column is-justify-content-center is-one-third" >}}
###### 综合分析工具集
依托数百种分析工具的全面组合，从数据中发掘深层价值。
###### 自动化分析工作流
可视化组合分析工具，用于创建易于复现的工作流。
###### 可扩展分析环境
探索由第三方分析工具构成丰富的插件生态系统，全面扩展功能。

{{< column-end >}} {{< columns-end >}}

{{< columns-start id="content-tab-4" >}} {{< column-start class="is-flex-direction-column is-two-thirds rounded mb-4" >}} ![分享地图](../img/share.gif "share map") {{< column-end >}}

{{<column-start class="is-flex-direction-column is-justify-content-center is-one-third" >}}
###### 业界领先的格式支持
凭借对各种数据源和格式的支持，成功应对数据集成挑战。
###### 标准与互操作性
通过探索、利用和创建基于行业标准的Web服务，赋能价值升级。
###### 成果发布
将QGIS拓展至云端与移动端，实现随时随地便捷访问。

{{< column-end >}} {{< columns-end >}}

### QGIS服务器
将您的QGIS工程和图层发布为符合OGC标准的WMS、WMTS、WFS及WCS服务。可自主控制要导出的图层、属性、布局和坐标系。QGIS服务器被视作 WMS 1.3 版本的参考实现。 {{< rich-content-end >}} {{< rich-box-end >}} {{< column-end >}} {{< column-start>}}

{{< rich-box-start >}} {{< rich-content-start themeClass="coloring-6" >}}
### QGIS桌面版
创建、编辑、可视化、分析与发布地理空间信息。

[下载]({{< ref "download.md" >}}) 为Windows、Mac、Linux、BSD和Android系统。 {{< rich-content-end >}} {{< rich-box-end >}} {{< column-end >}} {{< columns-end >}}
