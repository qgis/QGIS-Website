# Archivio
Le versioni precedenti di QGIS sono disponibili ai seguenti indirizzi:
* [Download ospitati da QGIS.org](/downloads)
* [Versioni precedenti per macOS / OS X](/downloads/macOS/)
* [Download ospitati da OSGeo](https://download.osgeo.org/qgis/)
* [Download legacy di "Kyngchaos" per macOS / OS X](https://www.kyngchaos.com/software/archive/).

Sono disponibili anche plugin per QGIS [all'indirizzo plugins.qgis.org](https://plugins.qgis.org/plugins/).
